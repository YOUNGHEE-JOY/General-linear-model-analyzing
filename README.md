# General linear model 
This is a statistical analysis report examining the impact of anxiety and community engagement on student academic performance. The report is divided into four sections to explain the data analysis process.
# Analysis plan
This section explains the hypotheses and the methods used to test them. It includes the rationale for choosing the GLM approach and how both main effects and interaction effects are examined.
# Coding
This section provides the specific code used for data analysis. It includes GLM statistical outputs, visualizations, assumption checks, and effect size calculations.
# Report
This section describes the main results in detail, supported by visual graphs and tables showing specific figures.
# Summary
This section reviews whether the hypotheses are supported and interprets the results in practical, real-world terms.
